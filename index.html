MUITO PRAZER, SOU SAMUEL MARTINS.

Sou um desenvolvedor Backend especializado em Java e Spring, com ampla experiência em projetos de desenvolvimento de software. Experiência em desenvolvimento de microserviços, integração com bancos de dados SQL e implementação de API RESTful. Domínio de metodologias ágeis, como Scrum e Kanban.

Destaco trabalhos recentes, como o Controle de Despesas, que envolveu o desenvolvimento de um sistema completo para gerenciamento de despesas com autenticação e autorização, e o projeto de microserviço de envio de e-mails, garantindo a entrega eficiente e segura de mensagens.
